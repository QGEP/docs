QGIS Server

You can download a VM with an older QGEP setup that also has QGIS Server installed from http://docs.qtibia.ro/qgisplatform.vdi.xz .

Acest VM conține instalări funcționale de QGIS Server și QGIS Web Client (QWC).

Accesare Servicii Web

The Apache server is setup to respond to requests pointing to the http://qgisplatform.demo. Accessing the above link with the Internet Browser of the VM will take you to the starting page of QWC depicted in the below image.

Dacă se dorește accesarea serviciilor Web din afara VM-ului, este necesară editarea fișierului hosts de pe stația de lucru de unde se vor accesa. În acest fișier va trebui făcută legătura dintre IP-ul VM-ului și numele acestuia, qgisplatform.demo.

Notă

Dacă adaptorul de rețea al VM-ului este setat sa funcționeze în mod NAT serviciile vor putea fi accesate numai de către gazdă/hipervizor. Pentru a putea accesa cu orice dispozitiv sau stație de lucru din rețeaua internă - LAN va trebui să setați adaptorul în mod bridge.
