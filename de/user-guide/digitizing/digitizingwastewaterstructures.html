3.2.2. Abwasserbauwerke digitalisieren

3.2.2.1. Allgemeines

QGEP verfügt über einen Assistenten, um Schächte und Sonderbauwerke einfach zu erfassen. Siehe dazu Kapitel Der QGEP Wizard (Zauberer).

Wähle den Wizard Knopf, klicke dann auf Start Data Entry und wähle dann Wastewater Structure im Dropdown-Menu

3.2.2.2. Digitalisieren

Der Cursor ändert zum Digitalisierungssymbol und es kann der Standort des neuen Punktelementes festgelegt werden.

Es öffnet das vw_gep_wastewater_structure Formular und die Daten können im Reiter Allgemein eingetragen werden:

Wähle den Abwasserbauwerk-Typ (vorgewählt ist ‚Normschacht‘):

  • Normschacht

  • Spezialbauwerk

  • Einleitstelle

  • Versickerungsanlage

Abhängig von ws_type gibt es unterschiedliche Felder und Reiter im Formular.

Fügen Sie dann die Bezeichnung (identifier) hinzu (dies ist das Attribut, das auf der Karte angezeigt wird).

Bemerkung

Wenn keine Bezeichnung eingegeben wird, übernimmt QGEP die Obj_ID als Bezeichnung (kann später geändert werden). Die Bezeichnung des Abwasserbauwerks ist auch der Vorgabewert für die Bezeichnung des Deckels und die Bezeichnung des Abwasserknotens.

Fügen Sie weitere Attributwerte im Reiter Allgemein hinzu. Sie können auch Attributwerte in anderen Reitern hinzufügen (Deckel, Abwasserbauwerk, Normschacht, Abwasserknoten).

Bemerkung

Die Idee hinter dem Reiter Allgemein ist, dass beim normalen Digitalisieren (95% der Normschächte) der Benutzer nicht zu einem anderen Reiter wechseln muss, um die notwendigen Felder abzufüllen.

Achtung

In den Reitern Deckel, Bauwerkteile, Abwasserknoten und Erhaltungsereignisse können keine neuen Datensätze hinzugefügt werden, weil zuerst der Abwasserbauwerk-Datensatz in der Datenbank gesichert werden muss. Die Aktionen können aus dem gleichen Grund auch noch nicht benutzen werden. Zuerst muss der Datensatz gesichert werden.

OK klicken um das Formular zu schliessen.

Speichern Sie die Informationen des Layers, indem Sie auf **Stop Data Entry ** klicken.

Sie können Ihr Punktobjekt bearbeiten, indem Sie den Bearbeitungsmodus auswählen und dann mit dem Info Tool auf das Objekt klicken, welches Sie bearbeiten möchten. Wenn Sie den Bearbeitungsmodus nicht auswählen, können Sie nur die existierenden Daten anschauen.

Detaillierte Informationen zum Bearbeiten siehe das Kapitel Bearbeiten von bestehenden Daten.

3.2.2.3. Weitere Attribute und Klassen

Wenn ein Abwasserbauwerk-Punktobjekt digitalisiert wird, findet eine Reihe von Schritten im Hintergrund in der QGEP Datenbank statt:

  1. ein neues Objekt Abwasserbauwerk wird hinzugefügt

  2. ein neues Objekt der entsprechenden Unterklasse [Normschacht, Spezialbauwerk, Versickerungsanlage, Einleitstelle] wird hinzugefügt und verknüpft

  3. ein neuer Deckel-Bauwerksteil wird hinzugefügt und mit dem Abwasserbauwerk verknüpft

  4. ein neuer Abwasserknoten wird in den Abwassernetzelementen und dessen Unterklasse Abwasserknoten erstellt

Wenn Sie zusätzliche Objekte (Bauwerkteile wie Einstiegshilfe, Bankett, Rückstauklappen usw.) hinzufügen wollen, dann müssen sie zuerst den Datensatz sichern und danach das Abwasserbauwerk editieren (vgl. das Kapitel Bearbeiten von bestehenden Daten ).

Bemerkung

Am einen zweiten/weiteren Deckel oder Abwasserknoten zum Abwasserbauwerk hinzuzufügen, vgl. Kapitel Bearbeiten von bestehenden Daten.

3.2.2.4. Geometrieabgleichung

Der digitalisierte Punkt wird auch für die verknüpften Klassen Deckel und Abwasserknoten übernommen. Der vw_qgep_Abwasserbauwerk-Punkt selber hat aber keinen Z-Wert. Wenn die Deckelkote ‚DE_Kote‘ erfasst wird, wird diese Kote als Z Wert bei der Deckelgeometrie verwendet. Der Wert aus dem Feld ‚AK_Sohlenkote‘ wird als Z Wert der Abwasserknotengeometrie übernommen.

Bemerkung

If a cover level changes, the Z value of the cover’s geometry will be adjusted. When the geometry changes, the co_level attribut is adjusted as well. If both values change, the level takes precedence. On an insert it’s like when both value change. Means the cover’s geometry is set according to the cover level and if it’s NULL, the Z value is set to NaN. The same situation is on editing the wastewater node directly.
