3.3.3. Hinzufügen oder bearbeiten von Erhaltungsereignissen

Eine Anleitung zum hinzufügen oder bearbeiten vom Erhaltungsereignissen Datensätzen in QGEP.

3.3.3.1. Allgemeines

  • Erhaltungsereignissen können mit einem oder mehreren Abwasserbauwerken verbunden sein. In der Datenbank ist das mit einer re-Zwischentabelle umgesetzt, um die n:m-Beziehung abzubilden (re_maintenance_event_wastewater_structure)

  • To avoid an error when adding a new Maintenance Event, you have to enable Buffered Transaction Groups in menu Project > Transaction mode

3.3.3.2. Ein neues Erhaltungsereignis hinzufügen

  • Aktivieren Sie den Bearbeitungsmodus.

  • Mit dem Objekte abfragen Werkzeug können Sie Schächte oder Haltungen auswählen indem sie den zugehörigen Layer auswählen (vw_qgep_Abwasserbauwerk oder vw_qgep_Haltung)

  • Das entsprechende Formular öffnet sich. Wechseln Sie zum Reiter Erhaltungsereignis .

  • Erstellen Sie ein neues Erhaltungsereignis mit dem Kindobjekt hinzufügen Knopf.

  • Bearbeite den neuen Datensatz im vw_qgep_maintenance Formular, dann klicke den OK Knopf

3.3.3.3. Verknüpfen mit einem bestehenden Erhaltungsereignis.

  • Aktivieren Sie den Bearbeitungsmodus.

  • Mit dem Objekte abfragen Werkzeug können Sie Schächte oder Haltungen auswählen indem sie den zugehörigen Layer auswählen (vw_qgep_Abwasserbauwerk oder vw_qgep_Haltung)

  • Das entsprechende Formular öffnet sich. Wechseln Sie zum Reiter Erhaltungsereignis .

  • Klicke den Vorhandene Kindobjekte verknüpfen Knopf.

  • Wähle ein Erhaltungsereignis aus dem Dialog Fenster mit Klick in in das Kästchen vor dem Datensatz. Das Kästchen muss gelb sein. Dann auf OK klicken

  • Schliessen Sie die Fenster mit OK und beenden Sie den Bearbeitungsmodus um die Änderungen zu speichern

3.3.3.4. Verknüpfung zu Erhaltungsereignissen auflösen

  • Aktivieren Sie den Bearbeitungsmodus.

  • Mit dem Objekte abfragen Werkzeug können Sie Schächte oder Haltungen auswählen indem sie den zugehörigen Layer auswählen (vw_qgep_Abwasserbauwerk oder vw_qgep_Haltung)

  • Das entsprechende Formular öffnet sich. Wechseln Sie zum Reiter Erhaltungsereignis .

  • Wähle ein Erhaltungsereignis mit Klick in das Kästchen vor dem Datensatz (das Kästchen wird gelb), dann klicke den Kindobjekt löschen Knopf.

Achtung

Es wird nur der Datensatz in der RE-Tabelle gelöscht. Der Erhaltungsereignis-Datensatz selbst wird nicht gelöscht. Er kann weiterhin mit anderen Abwasserbauwerken verknüpft werden.

3.3.3.5. Bearbeitung

  • Machen Sie es wie es beim Hinzufügen beschrieben - aber bearbeiten Sie nur die bereits bestehenden Erhaltungsereignisse. Oder bearbeiten Sie die vw_qgep_maintenance Tabelle (Layergruppe Inspektion )

3.3.3.6. Mehrfachbearbeitung

Zur Zeit gibt es kein Werkzeug um ein Erhaltungsereignis mit einer Auswahl von Haltungen oder Abwasserbauwerken zu verknüpfen. Wenn das Formular im Mehrfachbearbeiten-Modus geöffnet wird, ist der Reiter Erhaltungsereignis leer. Mehrfachbearbeiten mit verknüpften Tabellen ist zur Zeit in QGIS nicht umgesetzt.
