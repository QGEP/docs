QGIS Server

Sie können auch eine virtuelle Maschine mit einem älteren QGEP , welches auch einen QGIS Server installiert hat herunter laden, von: http://docs.qtibia.ro/qgisplatform.vdi.xz .

Die virtuelle Maschine hat funktionierende Installationen von dem QGIS Server <https://docs.qgis.org/latest/en/docs/user_manual/working_with_ogc/ogc_server_support.html>` und dem QGIS Web Client (QWC).

Betreten des Webservices

Der Apache-Server ist eingerichtet, um auf Anfragen an http://qgisplatform.demo zu reagieren. Bei Zugriff über vorstehende Verlinkung per Webbrowser, wird man durch die VM auf die Startseite von QWC geleitet wie untenstehend ersichtlich.

Wenn Sie den Webservice ausserhalb der VM betreten wollen, müssen Sie die hosts Datei auf Ihrer Maschine bearbeiten und qgisplatform.demo zur IP der VM richten.

Bemerkung

Wenn Sie die VM mit dem Netzwerk Adapter innerhalb des NAT Modus aufgesetzt haben, dann ist nur der VM host in der Lage auf die Services zuzugreifen. Wenn Sie wollen, dass alle Geräte in der LAge sind dies zu tun dann müssen Sie den Netzwerk Adapter im Bridge Modus aufsetzten.
