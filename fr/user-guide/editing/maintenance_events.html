3.3.3. Ajout et édition d’événements de maintenance

Ceci est un guide pour l’ajout ou l’édition de données relatives aux événements de maintenance dans QGEP.

3.3.3.1. Généralités

  • Maintenance events can be connected to one structure or to many. Therefore QGEP uses a intermediary RE_table to model the n:m relationship (re_maintenance_event_wastewater_structure)

  • To avoid an error when adding a new Maintenance Event, you have to enable Buffered Transaction Groups in menu Project > Transaction mode

3.3.3.2. Ajouter un nouvel incident de maintenance

  • Activez le mode d’édition.

  • Avec l’outil Identifier les entités, vous pouvez sélectionner des chambres ou des tronçons en sélectionnant leur couche respective (vw_qgep_wastewater_structure or vw_qgep_reach).

  • Le formulaire correspondant va s’ouvrir. Allez dans l’onglet Evénement de maintenance.

  • Créér un nouvel incident de maintenance avec le bouton Ajout entité-enfant.

  • Editez la nouvelle entrée dans le formulaire vw_qgep_maintenance, puis cliquez sur OK

3.3.3.3. Créer un lien avec un incident de maintenance existant

  • Activez le mode d’édition.

  • Avec l’outil Identifier les entités, vous pouvez sélectionner des chambres ou des tronçons en sélectionnant leur couche respective (vw_qgep_wastewater_structure or vw_qgep_reach).

  • Le formulaire correspondant va s’ouvrir. Allez dans l’onglet Evénement de maintenance.

  • Cliquez sur le bouton Lier entités-enfant existantes.

  • Choose a maintenance event from the Dialog window by clicking in the box of the record. The box must by yellow. Then click OK

  • Fermez la fenêtre en cliquant sur OK et désélectionnez le mode d’édition pour enregistrer les modifications.

3.3.3.4. Supprimer le lien avec incidents de maintenance

  • Activez le mode d’édition.

  • Avec l’outil Identifier les entités, vous pouvez sélectionner des chambres ou des tronçons en sélectionnant leur couche respective (vw_qgep_wastewater_structure or vw_qgep_reach).

  • Le formulaire correspondant va s’ouvrir. Allez dans l’onglet Evénement de maintenance.

  • Sélectionner un incident de maintenance en cliquant dans le case (la case devient jaune), puis cliquez sur le bouton Delete child feature

Attention

Seul la liaison stockée dans la RE-table est supprimé. Les événements de maintenance ne sont pas supprimés et peuvent toujours être reliés à d’autres ouvrages.

3.3.3.5. Édition

  • Faire comme décris dans le chapitre « Ajout » - mais en éditant des incidents de maintenance existant. Ou éditer la table vw_qgep_maintenance (groupe de couche inspection)

3.3.3.6. Multi Edit

Pour le moment, il n’existe pas d’outil permettant de lier un événement de maintenance avec une sélection de tronçons ou de structures assainissement. Si vous ouvrez le formulaire en mode multi-édition, l’onglet Maintenance sera vide. La multi-édition avec des tables liées n’est pas implémentée dans QGIS.
