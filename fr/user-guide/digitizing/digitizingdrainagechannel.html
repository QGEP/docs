3.2.4. Numériser une géométrie détaillée en tant que conduite de drainage

3.2.4.1. Généralités

You can add a polygon to a wastewater structure just with drawing a simple line. The function creates a rectangle from the line with a width of 20cm. This is specially useful for manholes with function drainage channel.

3.2.4.2. Outil de numérisation

  • Sélectionnez la couche vw_qgep_wastewater_structure

  • Faites un clic droit sur l’objet auquel vous voulez ajouter une canalisation de drainage

  • Faites un clic droit sur l’objet et sélectionnez la couche vw_qgep_wastewater_structure

  • Choisissez **Numériser conduite de drainage*

  • Vous pouvez maintenant commencer la numérisation: dessinez une ligne simple avec deux clics.

  • Le polygone est créé dans la couche od_ouvrage_assainissement

Une autre méthode consiste à ouvrir le formulaire vw_qgep_structure_assainissement avec l’outil info, puis de sélectionner Numériser une conduite de drainage dans le menu Actions du formulaire.
